Instructions
Étape 1: Créer un Token GitHub
  1. Allez sur GitHub Settings → Tokens
  2. Cliquez sur "Generate new token"
  3. Donnez un nom (ex: "VARICOM")
  4. Cochez "gist"
  5. Copiez le token généré
Étape 2: Configurer
  1. Importez votre fichier Excel
  2. Collez votre token GitHub ci-dessous
  3. Cliquez sur "Publier sur GitHub"
  4. Partagez le lien client généré
Ce token est stocké localement et n'est pas envoyé à nos serveurs
